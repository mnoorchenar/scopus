LaTeX Manuscript Analysis (NEW!)

Upload your LaTeX manuscript (.tex file) to enable advanced tracking:

  • Citation Frequency: How many times each reference is cited
  • Section Tracking: Which sections cite each reference
  • Section Filtering: View references by specific section (Introduction, Methods, etc.)
